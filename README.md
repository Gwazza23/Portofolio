# Portofolio

## Description

This portofolio was made as part of the portofolio project for Codecademy, the task was to build a static main page for a portofolio main page. Building this project helped me reinforce the basic coding language I had learnt previously. It also helped me with improving my planning and wireframing skills.
This project contains a main page and a contact page.

## Features

This project utilises HTML to build the static webpage, CSS to modify the stlyle of different elements and add responsivity to the webpage depending on the device being used and finally it uses JavaScript to add interactivity.
